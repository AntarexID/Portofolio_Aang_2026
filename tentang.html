Tentang Saya

Halo! Saya adalah seorang guru dan web developer yang tertarik pada dunia pendidikan, teknologi, dan kewirausahaan. Saya percaya bahwa teknologi dapat menjadi jembatan untuk menciptakan pembelajaran yang lebih efektif, kreatif, dan relevan dengan kebutuhan zaman.

Saya memiliki pengalaman dalam pengembangan website menggunakan HTML, CSS, dan JavaScript, serta terus mengembangkan kemampuan di bidang web development dan teknologi informasi selain itu saya mengikuti program MSIB Batch 3 di PT BISA ARTIFISIAL INDONESIA (BISA AI ACADEMY). Di dunia pendidikan, saya aktif berbagi pengetahuan dan membimbing peserta didik agar mampu memahami teknologi secara praktis dan aplikatif.

Selain itu, saya juga pernah mengikuti Pelatihan Kewirausahaan Dinas Tenaga Kerja Kabupaten Bandung, yang membekali saya dengan pemahaman tentang mindset kewirausahaan, ide dan peluang bisnis, perencanaan usaha, strategi pemasaran dan pemasaran digital, pengelolaan keuangan, serta etika dan hukum bisnis.

Melalui website ini, saya berbagi pengalaman, pengetahuan, dan karya yang saya kembangkan di bidang pendidikan, teknologi, dan web development. Saya terbuka untuk kolaborasi, pembelajaran, dan peluang baru yang memberikan dampak positif.
